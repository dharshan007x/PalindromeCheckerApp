## Description

The **Palindrome Checker App** is a simple Java console application that prints a welcome message for a palindrome checking system. This project serves as the basic starting point for a program that will eventually check whether a word or number is a palindrome.

A **palindrome** is a word, phrase, or number that reads the same forward and backward.
Examples: *madam, level, 121*

## Features

* Displays a welcome message for the application.
* Basic Java program structure.
* Can be extended to include palindrome checking functionality.

## Technologies Used

* Java
* Console-based application

## How to Run

1. Install Java (JDK) on your system.

2. Save the file as `PalindromeCheckerApp.java`.

3. Compile the program using:

   `javac PalindromeCheckerApp.java`

4. Run the program using:

   `java PalindromeCheckerApp`

## Expected Output
